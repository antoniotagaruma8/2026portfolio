<strong>Conditionals Master</strong> es una aplicación web educativa interactiva de alta fidelidad diseñada para dominar las estructuras condicionales en inglés. A través de atractivas mecánicas de gamificación y un contenido exhaustivo, la plataforma transforma el tema gramatical de las condicionales, a menudo desafiante, en una experiencia de aprendizaje dinámica e intuitiva.

El ciclo principal del juego—"¡Empareja el IF con el THEN!"—se complementa con modos de juego colaborativos y soporte lingüístico integrado.

<h2 className="section-label mt-12">Modos de Juego y Mecánicas</h2>

La aplicación presenta diversos modos gamificados adecuados para el estudio independiente y el aula:
<div className="space-y-8 my-8">
<div className="flex flex-col md:flex-row gap-2 md:gap-8 border-l-2 border-accent-blue/30 pl-4 md:pl-6 ml-2 md:ml-0 group">
  <div className="min-w-[220px] shrink-0 pt-1">
    <div className="text-sm font-bold text-foreground group-hover:text-accent-blue transition-colors">Modo Clásico</div>
  </div>
  <div className="text-sm text-muted-foreground leading-relaxed">
    Los alumnos conectan la cláusula "If" (Si) con su conclusión lógica "Then" (Entonces). Presenta progresión gamificada con barras de progreso dinámicas, contadores de rachas (🔥) e insignias multiplicadoras, junto con feedback estilo arcade (puntos flotantes, confeti).
  </div>
</div>
<div className="flex flex-col md:flex-row gap-2 md:gap-8 border-l-2 border-accent-blue/30 pl-4 md:pl-6 ml-2 md:ml-0 group">
  <div className="min-w-[220px] shrink-0 pt-1">
    <div className="text-sm font-bold text-foreground group-hover:text-accent-blue transition-colors">Story Sparks</div>
  </div>
  <div className="text-sm text-muted-foreground leading-relaxed">
    Un modo colaborativo perfecto para la práctica oral en clase. Los estudiantes terminan oraciones usando vocabulario proporcionado ("sparks") o su propia imaginación, creando una cadena de historias en la que el siguiente estudiante comienza su oración usando la consecuencia anterior.
  </div>
</div>
</div>

<h2 className="section-label mt-12">Arquitectura Técnica</h2>

El proyecto está construido como una aplicación web vanilla ligera y altamente responsiva, asegurando máxima compatibilidad y tiempos de carga rápidos.
<div className="space-y-8 my-8">
<div className="flex flex-col md:flex-row gap-2 md:gap-8 border-l-2 border-accent-blue/30 pl-4 md:pl-6 ml-2 md:ml-0 group">
  <div className="min-w-[220px] shrink-0 pt-1">
    <div className="text-sm font-bold text-foreground group-hover:text-accent-blue transition-colors">Frontend y Estilos</div>
  </div>
  <div className="text-sm text-muted-foreground leading-relaxed">
    Utiliza HTML5, CSS3 y Vanilla JavaScript con un UI/UX moderno que presenta glassmorfismo, transiciones suaves, insignias en forma de píldora y fuentes de Google personalizadas (`Nunito`). Diseño totalmente responsivo adaptado para móviles, tablets y ordenadores de sobremesa.
  </div>
</div>
<div className="flex flex-col md:flex-row gap-2 md:gap-8 border-l-2 border-accent-blue/30 pl-4 md:pl-6 ml-2 md:ml-0 group">

<h2 className="section-label mt-12">Impacto y Casos de Uso</h2>

Conditionals Master acorta la brecha entre la memorización gramatical de memoria y el uso activo y atractivo. Para los profesores, sirve como una herramienta lista para usar en sesiones de pizarra interactiva. Para los estudiantes, proporciona una plataforma visualmente gratificante para practicar estructuras gramaticales complejas hasta que se vuelvan intuitivas. Como proyecto de desarrollo, se erige como una muestra limpia de la gestión de estado de Vanilla JavaScript y el renderizado dinámico de la interfaz de usuario sin la sobrecarga de frameworks pesados.
