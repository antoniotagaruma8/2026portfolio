<div className="text-base text-muted-foreground leading-relaxed max-w-3xl mb-12">
El proyecto del <strong>Sistema Educativo de Filipinas</strong> es una presentación interactiva, creada a medida y basada en la web, diseñada para ofrecer una visión integral del panorama educativo, la cultura y las estructuras sistémicas en Filipinas. Alejándose de las tradicionales plataformas de diapositivas (como PowerPoint o Google Slides), este proyecto aprovecha las tecnologías web para crear una herramienta educativa dinámica, altamente personalizable y visualmente atractiva.

Proporciona una mirada profunda a la revolución K-12 del país, las políticas lingüísticas bilingües, la cultura escolar profundamente arraigada, la vida estudiantil y los desafíos únicos que enfrenta el sistema, diseñados específicamente para una audiencia internacional.
