<div className="text-base text-muted-foreground leading-relaxed max-w-3xl mb-12">
<strong>Echo News</strong> es una aplicación web inmersiva y gamificada diseñada para ayudar a los estudiantes de inglés de nivel B2 a dominar las complejidades del Estilo Indirecto (Reported Speech). Construida como una experiencia de "Aula Arcade", la plataforma transforma los ejercicios de gramática tradicionales en misiones narrativas de alto riesgo a través de varios escenarios profesionales y encubiertos, desde interceptar comunicaciones de espías hasta informar sobre dramas judiciales y negociaciones en la sala de juntas.

La aplicación combina a la perfección el rigor educativo con un diseño de interfaz de usuario oscuro y atractivo (UI/UX), utilizando un motor de juego a medida y multimodal que evalúa a los estudiantes en el retroceso de tiempos verbales (backshifting), ajustes de pronombres/tiempo, y el uso preciso de verbos de reporte avanzados.
</div>

<h2 className="section-label mt-12">Mecánicas de Gamificación e Interactividad</h2>

La plataforma cuenta con un motor de escenarios múltiples con distintos modos de juego especializados para apuntar a diferentes habilidades cognitivas y lingüísticas:
<div className="space-y-8 my-8">
<div className="flex flex-col md:flex-row gap-2 md:gap-8 border-l-2 border-accent-blue/30 pl-4 md:pl-6 ml-2 md:ml-0 group">
  <div className="min-w-[220px] shrink-0 pt-1">
    <div className="text-sm font-bold text-foreground group-hover:text-accent-blue transition-colors">Modos de Juego Especializados</div>
  </div>
  <div className="text-sm text-muted-foreground leading-relaxed">
    Incluye <strong>Classic Mode</strong> (opción múltiple), <strong>Scramble Mode</strong> (construcción de sintaxis de arrastrar y soltar), <strong>Proofreader Mode</strong> (corrección de textos), y <strong>Narrative/Interrogator Mode</strong> (simulación dinámica de historial de chat).
  </div>
</div>
<div className="flex flex-col md:flex-row gap-2 md:gap-8 border-l-2 border-accent-blue/30 pl-4 md:pl-6 ml-2 md:ml-0 group">
  <div className="min-w-[220px] shrink-0 pt-1">
    <div className="text-sm font-bold text-foreground group-hover:text-accent-blue transition-colors">Wiretap Mode (Audio)</div>
  </div>
  <div className="text-sm text-muted-foreground leading-relaxed">
    Un modo de comprensión auditiva avanzado que utiliza la API de Síntesis de Voz Web. Los estudiantes escuchan audios "interceptados" y deben informar con precisión lo que escucharon, conectando las habilidades auditivas con la producción gramatical.
  </div>
</div>
<div className="flex flex-col md:flex-row gap-2 md:gap-8 border-l-2 border-accent-blue/30 pl-4 md:pl-6 ml-2 md:ml-0 group">
  <div className="min-w-[220px] shrink-0 pt-1">
    <div className="text-sm font-bold text-foreground group-hover:text-accent-blue transition-colors">Escenarios Dinámicos</div>
  </div>
  <div className="text-sm text-muted-foreground leading-relaxed">
    Para evitar la fatiga y proporcionar variedad contextual, la aplicación ofrece diversas misiones temáticas, como Echo News (Periodismo), Precinct 101 (Interrogatorio policial), Spilled Tea (Chismes sensacionalistas), Boardroom Brief (Finanzas corporativas), Witness Stand (Drama judicial), Secret Agent (Operaciones encubiertas), y Sideline Report (Análisis deportivo).
  </div>
</div>
</div>

<h2 className="section-label mt-12">Arquitectura Técnica y UI/UX</h2>

La aplicación está construida utilizando un stack web vanilla ligero y modular, lo que garantiza un alto rendimiento, cero dependencias y una amplia compatibilidad.
<div className="space-y-8 my-8">
<div className="flex flex-col md:flex-row gap-2 md:gap-8 border-l-2 border-accent-blue/30 pl-4 md:pl-6 ml-2 md:ml-0 group">
  <div className="min-w-[220px] shrink-0 pt-1">
    <div className="text-sm font-bold text-foreground group-hover:text-accent-blue transition-colors">Gestión de Datos y Estado</div>
  </div>
  <div className="text-sm text-muted-foreground leading-relaxed">
    Una robusta arquitectura de datos similar a JSON (`data.js`) alberga cientos de preguntas de gramática meticulosamente elaboradas. Un motor de juego centralizado (`script.js`) maneja el enrutamiento de misiones, el seguimiento de puntajes, la mezcla de arreglos (Fisher-Yates) para una rejugabilidad aleatoria y la inyección dinámica del DOM.
  </div>
</div>
<div className="flex flex-col md:flex-row gap-2 md:gap-8 border-l-2 border-accent-blue/30 pl-4 md:pl-6 ml-2 md:ml-0 group">
  <div className="min-w-[220px] shrink-0 pt-1">
    <div className="text-sm font-bold text-foreground group-hover:text-accent-blue transition-colors">Integración de Audio</div>
  </div>
  <div className="text-sm text-muted-foreground leading-relaxed">
    Un contenedor de la API de Web Audio personalizado (`sfx.js`) administra los efectos de sonido inmersivos de la interfaz de usuario (navegación, comentarios correctos/incorrectos, revelaciones de misiones) y se vincula con el motor de texto a voz (TTS) para el modo Wiretap.
  </div>
</div>
<div className="flex flex-col md:flex-row gap-2 md:gap-8 border-l-2 border-accent-blue/30 pl-4 md:pl-6 ml-2 md:ml-0 group">
  <div className="min-w-[220px] shrink-0 pt-1">
    <div className="text-sm font-bold text-foreground group-hover:text-accent-blue transition-colors">Estética de "Cibernoticiario"</div>
  </div>
  <div className="text-sm text-muted-foreground leading-relaxed">
    La estética está diseñada deliberadamente para sentirse como una terminal premium de alto riesgo. Utiliza glassmorfismo, acentos brillantes, un modo oscuro y microinteracciones (animaciones de sacudida para errores, visualizaciones dinámicas de formas de onda) para mantener al usuario visualmente comprometido.
  </div>
</div>
</div>
