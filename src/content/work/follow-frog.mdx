<div className="text-base text-muted-foreground leading-relaxed max-w-3xl mb-12">
<strong>Follow the Frog</strong> is a streamlined, gamified web application tailored for ESL/EFL learning environments. It leverages multi-modal inputs—beginning with an audio-only listening challenge and culminating in a video reveal—to build auditory processing skills, focus, and narrative prediction. 

The project features a custom Web Audio API visualizer that responds to the audio input in real-time, creating an engaging, sensory-rich experience for students before they transition to the final visual context.
</div>

<h2 className="section-label mt-12">Technical Architecture</h2>

The application is built as a lightweight, client-side single-page application (SPA) using vanilla web technologies, ensuring maximum compatibility and fast load times.
<div className="space-y-8 my-8">
<div className="flex flex-col md:flex-row gap-2 md:gap-8 border-l-2 border-accent-blue/30 pl-4 md:pl-6 ml-2 md:ml-0 group">
  <div className="min-w-[220px] shrink-0 pt-1">
    <div className="text-sm font-bold text-foreground group-hover:text-accent-blue transition-colors">Real-Time Audio Visualization</div>
  </div>
  <div className="text-sm text-muted-foreground leading-relaxed">
    Utilizes the `AudioContext` and `AnalyserNode` APIs in tandem with HTML5 `<canvas>` to render a dynamic frequency bar graph, programmatically coloring the bars with frog-themed RGB values.
  </div>
</div>
<div className="flex flex-col md:flex-row gap-2 md:gap-8 border-l-2 border-accent-blue/30 pl-4 md:pl-6 ml-2 md:ml-0 group">
  <div className="min-w-[220px] shrink-0 pt-1">
    <div className="text-sm font-bold text-foreground group-hover:text-accent-blue transition-colors">Media Handling</div>
  </div>
  <div className="text-sm text-muted-foreground leading-relaxed">
    Native HTML5 `<audio>` and `<video>` elements are manipulated via JavaScript event listeners (like `play` and `onended`) to control the application flow and coordinate the sequential reveal.
  </div>
</div>
<div className="flex flex-col md:flex-row gap-2 md:gap-8 border-l-2 border-accent-blue/30 pl-4 md:pl-6 ml-2 md:ml-0 group">

  </div>
  <div className="text-sm text-muted-foreground leading-relaxed">
    Employs a cohesive "frog" color palette with cyan backgrounds and vibrant green call-to-action buttons. CSS Flexbox centers the application within a sleek, responsive card with smooth state transitions.
  </div>
</div>
